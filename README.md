# SCT_DA_3

## Interactive Visualization using Power BI

### Objective
Create an interactive dashboard to analyze employee attrition and identify factors contributing to employee turnover.

### Dataset
IBM HR Employee Attrition Dataset

### Dashboard Features
- Department Filter
- Age Group Filter
- Gender Filter
- Attrition by Department
- Attrition by Age Group
- Overtime and Attrition Analysis
- Job Satisfaction vs Attrition

### Key Insights
1. Research & Development and Sales departments show higher attrition.
2. Employees aged 26–35 experience the highest turnover.
3. Employees working overtime are more likely to leave.
4. Job satisfaction impacts employee retention.
5. Employee attrition varies across departments and demographic groups.

### Tools Used
- Power BI
- GitHub

### Outcome
The dashboard provides interactive analysis to understand why employees leave the organization and helps identify key factors affecting employee retention.# SCT_DA_3
